# Starting the Server
### 1. Get Upstream Updates
`git fetch upstream`

### 2. Create .env
* in most files need a .env file
* create .env file in root folder of your fork
* paste this text and save

### 3. Start Server
`yarn start`

### 4. Check scripts
> check package.json for scripts etc.
 

# Reset Branch

`git status`

`git checkout (file)`

`git checout src (all)`

`git reset --hard upstream/master`
