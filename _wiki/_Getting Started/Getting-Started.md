### Generating SSH Key (One-time step)
This is a one time step UNLESS im accessing AP or EQX

* `$ ssh-keygen -t rsa`

* `cd ~/.ssh`

* `ls -la`

* `pbcopy < ~/.ssh/id_rsa.pub`

### Installing Brew
`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"`

`brew install yarn`

`brew list`
